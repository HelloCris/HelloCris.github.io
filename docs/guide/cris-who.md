# 关于 Cris

## 👋 个人介绍

你好！我是 Cris，一名热爱技术、追求简洁与体验的前端开发者，也是这份学习笔记的作者与长期维护者。

我的前端之旅始于大四——那时的我，面对浏览器中的代码与像素，第一次感受到“创造”的乐趣。硕士期间，我系统自学了 HTML、CSS、JavaScript 及现代前端框架，并通过大量实践项目不断打磨工程化思维与解决问题的能力。

2022年毕业后，我正式踏入前端行业。第一次实习，是在深圳顺丰科技做saas服务。第一份工作，是在南京荣耀手机做海外商城开发。现在回到了老家，在青岛海尔依然从事着商城前端开发。我的履历都是toC项目，因此我一直秉承着：“前端不仅是界面的实现者，更是用户体验的守护者与性能的优化者”的信念。

这份笔记建立的初衷只是觉得自己学习笔记格式繁杂，有word、pdf、md、截图等等，想整理一下。转念一想，我作为前端工作者，自己用低成本搭建一个服务写笔记也未尝不可。虽然刚刚起步，但希望它能逐渐成长为一个持续迭代的技术仓库。记录我的思考、踩过的坑、学到的新范式，也见证我从“能跑就行”到“优雅可靠”的成长轨迹。
